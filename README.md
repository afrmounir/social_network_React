## Getting Started

### Prerequisites

- Node 16.16.0
- A MongoDB database

### Installation

1. Launch the <a href="https://github.com/afrmounir/social_network_REST_API">API</a>
2. Clone the repo
   ```sh
   git clone git@github.com:afrmounir/social_network_React.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Run the project
   ```sh
   npm start
   ```
5. Open 'http://localhost:3000' in Chrome
